# Food Order Console App (Java)

A simple console-based food ordering system written in Java.

## Features
- Add new food orders
- View orders by ID
- Edit orders (only within 10 minutes)
- Delete orders
- Set delivery date and time

## Technologies
- Java
- Scanner
- ArrayList
- LocalDateTime

## How to Run
1. Open project in IntelliJ
2. Run `Main.java`
3. Follow console instructions
